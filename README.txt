上传方法：
1. 在 GitHub 新建公开仓库 June618.github.io
2. 点击 Add file → Upload files
3. 把你电脑里的个人照片重命名为 profile.jpg
4. 将 index.html 和 profile.jpg 一起拖入上传框
5. 点击 Commit changes
6. 打开仓库 Settings → Pages，选择 Deploy from a branch、main、/(root)，然后 Save
7. 等待约 1–10 分钟后访问 https://June618.github.io/

注意：只有当你的 GitHub 用户名是 June618 时，这个用户主页地址才会生效。
